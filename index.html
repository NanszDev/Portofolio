Kemampuan saya

Saya memiliki kemampuan dalam pengembangan website menggunakan HTML, CSS, dan JavaScript untuk membuat tampilan modern, responsif, dan interaktif.

HTML (HyperText Markup Language) adalah bahasa markah standar yang digunakan untuk membuat dan menyusun struktur sebuah halaman website

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mendesain, mengatur gaya, dan mempercantik tampilan visual dokumen (seperti halaman web) yang ditulis dalam bahasa markup seperti HTML.

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan untuk membuat halaman web menjadi lebih dinamis dan interaktif

Python adalah bahasa pemrograman tingkat tinggi yang serbaguna, populer, dan mudah dipelajari. Dikenal karena sintaksnya yang bersih dan mirip bahasa manusia,
